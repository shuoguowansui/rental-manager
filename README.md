# 🏗️ 出租管理系统 v2

脚手架、围挡、钢管架出租管理工具。

## 功能

- 📊 仪表盘：订单数、库存、营收概览
- 📦 库存管理：脚手架/围挡/钢管架三品类
- 📋 订单管理：新增、归还、搜索筛选
- 💰 付款追踪：已付/未付状态
- 📈 记账系统：月度分析、排名、日统计
- 📱 PWA：可安装到手机，离线可用

## 技术

HTML/CSS/JS 单文件 + localStorage + Service Worker + PWA
